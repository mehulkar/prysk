Dependencies#

In order to run all tests locally you need to have the following tools installed.

Python#

  • python >= 3.7

  • poetry

Shells#

  • dash

  • bash

  • zsh

If you have these dependencies all setup, just run a

poetry install

within the root folder of the project. Now you should be good to go!

Nox#

Mostly all task you will need to take care of are automated using nox. So if you want to run all checks and build the documentation etc. just run:

nox

To get a list of all available targets run:

nox --list

For running a specific target run:

nox -s <target>
